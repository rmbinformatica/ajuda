# cmd & batch script

## Listagem de diretório atual

```text
dir
```

## Mudando de diretório

```text
cd [caminho]
```

## Adicionando credenciais de acesso a um servidor de rede no cofre do usuário

```text
cmdkey /add:[nome_ou_ip] /user:[username] /pass:[senha]
```
